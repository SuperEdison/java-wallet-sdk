# web3-chain-btc

比特币链适配器（预留占位）。

## 状态

🚧 **开发中**

## 计划支持

| 功能 | 说明 | 状态 |
|------|------|------|
| P2PKH | Legacy 地址 (1xxx) | 🚧 |
| P2WPKH | SegWit 地址 (bc1q) | 🚧 |
| P2TR | Taproot 地址 (bc1p) | 🚧 |
| ECDSA | secp256k1 签名 | 🚧 |
| Schnorr | Taproot 签名 | 🚧 |

## HD 路径

- BIP-84 (SegWit): `m/84'/0'/0'/0/0`
- BIP-86 (Taproot): `m/86'/0'/0'/0/0`
